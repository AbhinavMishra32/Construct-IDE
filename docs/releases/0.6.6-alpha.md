# Construct 0.6.6-alpha

## Changes

- Construct Cloud model selection now loads the Cloud catalog automatically when the Cloud source is selected or its token changes.
- The AI settings model selector no longer uses the legacy rich provider picker for the Construct agent model; it uses the Opaline select surface and keeps Construct Cloud choices catalog-owned.
- The Concepts graph keeps the Construct loading shimmer visible until the graph engine reports readiness, avoiding a blank knowledge-web pane during startup.
- Flow chat unlocks from terminal session state even when a completed, errored, or waiting session arrives through an update event, preventing stale Thinking/send loading states after normal model completion.
- GitHub release publishing now honors the workflow's explicit prerelease flag, so tag-pushed alpha builds can publish as normal releases when the workflow sets `IS_PRERELEASE=false`.
