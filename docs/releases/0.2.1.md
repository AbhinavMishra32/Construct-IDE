# Construct 0.2.1

Construct 0.2.1 rebuilds Construct Interact around live agent activity and compiler-validated Dynamic Steps.

## Highlights

- Tool calls, results, reasoning activity, and response text now appear while the run is happening
- Completed activity remains expandable without duplicating a final response inside the trace
- Removed the fixed 180-second whole-run timeout that discarded valid long-running agent work
- Confidence and advancement are now recorded only through an explicit learner-assessment tool
- Dynamic Steps are created by reusable tape tools instead of being embedded in the model's final JSON
- New tools inspect the tape overview, exact authored steps and blocks, and authored file manifests
- Proposed Dynamic Steps can call the real parser and compiler before creation
- Dynamic Steps support authored tape blocks including explain, guide, interact, edit, recall, run, expect, and checkpoint
- Dynamic Step actions open the generated step directly and render block-by-block in the Guide
- Construct Interact and right-side project panels now have reliable internal scrolling
- Added expandable tool input/result details, streaming response caret, trace transitions, and Dynamic Step motion
- Project Advanced settings can edit and validate the actual `.construct` tape
