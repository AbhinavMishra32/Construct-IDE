# Construct 0.0.3

Construct `0.0.3` is the first official desktop release cut for the new tape-first Construct app.

## What shipped

- A unified Construct desktop brand with the new app icon across the app, website, and packaged assets.
- Cross-platform packaging configuration for macOS, Windows, and Linux through command-based release scripts.
- A tape runtime centered on backwards-compatible support through `tape-0.3.1`.
- A sharper desktop editing experience with improved terminal polish, process debugging surfaces, and agent-runtime abstraction work.

## Why this release matters

Construct is now organized around one clear product surface: a desktop IDE for executable learning tapes.

That means:

- the old pre-tape runner architecture is out
- the release path is scriptable without GitHub Actions
- the desktop app, website, and release assets share one brand system
- older tapes remain usable while the spec evolves

## Release assets

This release is designed to publish:

- macOS: `.dmg`, `.zip`
- Windows: `nsis`, `portable`, `.zip`
- Linux: `AppImage`, `.deb`, `.tar.gz`

## Notes

- GitHub publication is handled through the `gh` CLI.
- Packaging is command-based so the same flow can run locally or inside any CI provider.
- Older tape projects remain a compatibility target, not a migration casualty.
