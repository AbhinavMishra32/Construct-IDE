# Construct 0.0.3

Construct `0.0.3` is the first release pass aimed at real desktop downloads.

## Highlights

- Packaged Electron app targets for macOS, Windows, and Linux.
- Unified app branding with the new Construct icon.
- Tape runtime kept backwards-compatible through `tape-0.3.1`.
- Sharper editor and terminal runtime polish inside the desktop app.

## Release Assets

- macOS: `.dmg`, `.zip`
- Windows: `nsis`, `portable`, `.zip`
- Linux: `AppImage`, `.deb`, `.tar.gz`

## Notes

- GitHub publication is handled with the `gh` CLI instead of GitHub Actions.
- Packaging is command-based so any CI provider can run the same release steps.
