# 0.6.4-alpha

Construct 0.6.4-alpha is a prerelease that finishes the Windows shell and branding polish pass while keeping macOS behavior unchanged.

## What's Changed

- Refined the Windows shell radius and acrylic clipping so the app chrome feels rounder and more native
- Kept Windows drag, shadow, and acrylic behavior in place while removing the awkward transparent-sheet artifacts
- Reworked the empty Construct agent panel to use the Construct logo and center itself vertically in the session surface
- Renamed visible agent copy from Flow to Construct across the desktop UI, including project actions, model selection, and composer prompts
- Marked imported `.construct` tape surfaces as legacy tape in the visible UI while preserving the underlying project format
- Restyled the Construct Cloud unreachable state to match the sign-in aesthetic and removed the dark-mode glow
- Added and updated renderer tests to lock the new branding and shell copy in place
- Bumped the desktop release metadata to `0.6.4-alpha`

## Highlights

- Windows now has the latest shell polish pass with rounder corners and cleaner chrome
- The main project agent is now surfaced as Construct agent in the UI
- Legacy tape projects remain supported, but the UI now calls them out as legacy
