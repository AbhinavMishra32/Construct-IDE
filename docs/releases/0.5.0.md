# Construct 0.5.0

Construct 0.5.0 focuses on release readiness, Flow workspace responsiveness, and a more coherent concept-learning surface.

## Highlights

- Bottom terminal opens reliably from the panel and shortcut path, with visible-aware xterm mounting and rAF resize fitting
- Panel and tree interactions are instant and closer to VS Code behavior through the updated Opaline shell
- Flow project opening avoids broad project refreshes so inactive dashboard/home data is not pulled into the hot path
- File explorer adds a bottom fade affordance for scrollable trees
- Concepts now default to the active Flow project instead of showing the global knowledge base first
- Concepts has a card-based browser with current-project/all-concepts switching and stronger project relation metadata
- Knowledge web adds a lazy-loaded 3D force graph for parent and related concept links
- CI and release checkout only the public Opaline submodule and no longer recursively fetch the private backend
- Release preflight now checks Opaline submodule URL, workflow checkout behavior, release asset naming, and version alignment
