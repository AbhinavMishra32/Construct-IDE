# Construct 0.6.1

Construct 0.6.1 is a release-notes and presentation refresh off the `main` branch. The mainline delta since `v0.6.0` is intentionally small and focused on the public-facing story rather than runtime behavior.

## Highlights

- README copy was rewritten to better frame Construct as a learning IDE instead of a generic coding agent
- The landing section now points readers to the newer `0.6.1` release notes and version badge
- Product screenshots were refreshed to keep the public docs aligned with the current UI surface
- The release tag now tracks the same root/app version pair, so packaging metadata stays aligned at `0.6.1`
